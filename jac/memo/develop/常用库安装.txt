Linux环境下
STLPort
1. 修改stlport/stl/type_traits.h

    需要包含这个 boost/type_traits/remove_const.hpp

    请在stlport/stl/type_traits.h 

    #ifdef _STLP_USE_BOOST_SUPPORT
    # include <stl/boost_type_traits.h>
    # include <boost/type_traits/add_reference.hpp>
    # include <boost/type_traits/add_const.hpp>
    # include <boost/type_traits/remove_const.hpp>    //新添加的
    #endif /* _STLP_USE_BOOST_SUPPORT */

2. 根目录下./configure --prefix=/home/jacteefan/CPP/STLport-5.2.1 --with-boost=/home/jacteefan/CPP/boost_1_46_0 --with-static-rtl  --enable-static 
3. 进入build/lib
4. make clean install

boost

    using gcc;
    using stlport : 5.2.1 : /home/jacteefan/CPP/STLport-5.2.1 : /home/jacteefan/CPP/STLport-5.2.1/lib ;   
 2. ./bootstrap.sh --prefix=/home/jacteefan/CPP/boost_1_46_0
 3.  ./bjam stdlib=stlport install

 windows环境下

STLport
1. 同上
2. 根目录下configure msvc8 --with-static-rtl --use-boost 目录
3. 进入build/lib
